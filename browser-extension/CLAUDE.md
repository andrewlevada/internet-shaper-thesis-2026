## Fiber

This chrome extension is built with [fiber-extension](../fiber/README.md) framework. It is not yet published on npm — we are using a local version.

```bash
cd browser-extension
pnpm dev    # Dev mode with HMR
pnpm build  # Production build
```

You may adjust, fix and improve fiber while developing this extension. After changing it's code rebuild to see updates.
